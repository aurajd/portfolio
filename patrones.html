Afrontar ejercicios Acepta el reto

  1. Análisis
    • Reconocer datos de entrada y de salida.
  2. Ejemplos
    • Crear casos de ejemplo, señalando los datos de entrada a utilizar y los datos de salida esperados.
  3. Andamio
    • Creamos la base del programa, sin necesidad de que funcione por ahora.
  4. Parseo
    • Adaptamos los datos de entrada a lo requerido
  5. Programar cada ejemplo
    • Programamos para que nuestra aplicación sea capaz de resolver todos nuestros ejemplos.
